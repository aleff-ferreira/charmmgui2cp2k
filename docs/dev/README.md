# Developer / methodology notes (not part of the public tool)

This directory holds internal working material kept out of the public top level:

- `PUBLICATION_READINESS_PLAN.md` — the rigor/validation/usability roadmap.
- `AGENTS.md` — autonomous-agent operating instructions used during development.
- `prompt_kit/`, `prompt_kit_tui/` — phase-gated prompt kits.

None of it is required to build, run, test, or cite `charmmgui2cp2k`. It may be
deleted before a public release (`git rm -r docs/dev/`); it is retained here for
provenance and reproducibility of the development process.
